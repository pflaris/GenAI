# Tema e delimitação

    Para iniciar o trabalho, foi dado um contexto da situação de forma a "preparar" a inteligência da forma que gostaríamos que ela se comportasse:

#### *Prompt de preparo*
**`Sou um aluno de graduação em Sistemas para Internet e estou desenvolvendo minha pesquisa científica para o TCC. Preciso que você me ajude durante o processo, me dando respostas diretas e detalhadas. Você consegue dar as respostas em tópicos para evitar respostas abrangentes?`**
    

O segundo prompt enviado já foi mais direto, a fim de receber uma resposta objetiva para a pergunta.
#### *Primeira interação*


**`Sou graduando em Sistemas para Internet e tenho interesse em Inteligência Artificial. Liste 3 temas de pesquisa relevantes e atuais `**
    
| **Tema** | **Descrição** |
|----------------|----------------|
| *Aplicação de Modelos de IA para Personalização da Experiência do Usuário na Web* | Exploração do uso de machine learning para personalizar conteúdos, recomendações e interfaces adaptativas em sites e aplicativos. Análise dos impactos na usabilidade, retenção de usuários e conversão em plataformas digitais. |
| *Uso de IA para Detecção e Prevenção de Ataques Cibernéticos em Aplicações Web* |  Desenvolvimento de sistemas inteligentes para identificar padrões de ataques como SQL Injection, Cross-Site Scripting (XSS) e ataques de negação de serviço (DDoS). Comparação entre diferentes algoritmos de detecção e suas taxas de precisão e eficiência. |
| *Geração de Código Automatizada com IA para Desenvolvimento Web* |Estudo sobre o impacto de ferramentas como GitHub Copilot e ChatGPT no desenvolvimento web. Avaliação da qualidade do código gerado, produtividade dos desenvolvedores e desafios éticos e de direitos autorais  |


#### *Segunda interação*
**`Delimite o tema Aplicação de Modelos de IA para Personalização da Experiência do Usuário na Web, Uso de IA para Detecção e Prevenção de Ataques Cibernéticos em Aplicações Web, Geração de Código Automatizada com IA para Desenvolvimento Web, colocando os elementos o que, quem, onde, quando, como e por quê.`**

| Aplicação de Modelos de IA para Personalização da Experiência do Usuário na Web |  |
| :---- | :---- |
| **O que?** | Aplicação de algoritmos de inteligência artificial para personalizar a experiência do usuário em sites e aplicações web, com base no comportamento, preferências e interações do usuário. |
| **Quem?** | Usuários de plataformas digitais, desenvolvedores web, empresas de tecnologia e pesquisadores em IA. |
| **Onde?** | Em plataformas de e-commerce, redes sociais, serviços de streaming, sistemas de recomendação e aplicações web interativas. |
| **Quando?** | No momento da interação do usuário com a plataforma, utilizando aprendizado de máquina em tempo real ou baseado em histórico de navegação. |
| **Como?** | Por meio de algoritmos de machine learning, deep learning e processamento de linguagem natural (PLN), que analisam padrões de comportamento e ajustam conteúdos, interfaces e recomendações personalizadas. |
| **Por quê?** | Para aumentar o engajamento, retenção e satisfação do usuário, melhorando a usabilidade e otimizando a entrega de conteúdos relevantes. |

| Uso de IA para Detecção e Prevenção de Ataques Cibernéticos em Aplicações Web |  |
| :---- | :---- |
| **O que?** | Desenvolvimento e implementação de modelos de inteligência artificial para identificar, prevenir e mitigar ataques cibernéticos em aplicações web. |
| **Quem?** | Administradores de segurança da informação, empresas de tecnologia, desenvolvedores web, pesquisadores de cibersegurança e usuários que acessam plataformas online. |
| **Onde?** | Em aplicações web, servidores, redes corporativas e ambientes de nuvem que hospedam sistemas vulneráveis a ataques como SQL Injection, Cross-Site Scripting (XSS) e ataques de negação de serviço (DDoS). |
| **Quando?** | Durante a navegação e interação com aplicações web, monitorando tráfego de dados em tempo real ou por meio de análise preditiva baseada em dados históricos de ataques. |
| **Como?** | Utilizando algoritmos de machine learning, análise comportamental, redes neurais e sistemas de detecção de intrusão (IDS) que identificam padrões suspeitos e bloqueiam ameaças antes que causem danos. |
| **Por quê?** | Para fortalecer a segurança de aplicações web, reduzir vulnerabilidades, evitar prejuízos financeiros e proteger dados sensíveis contra invasões maliciosas. |

| Geração de Código Automatizada com IA para Desenvolvimento Web |  |
| :---- | :---- |
| **O que?** | Utilização de modelos de inteligência artificial para auxiliar no desenvolvimento de código-fonte, otimizando a escrita, correção e refatoração de software para aplicações web. |
| **Quem?** | Desenvolvedores web, engenheiros de software, empresas de tecnologia e pesquisadores em IA aplicada ao desenvolvimento. |
| **Onde?** | Em ambientes de desenvolvimento integrado (IDEs), plataformas de versionamento de código (como GitHub) e ferramentas de apoio à programação, como GitHub Copilot, ChatGPT e Codeium. |
| **Quando?** | Durante a fase de codificação, manutenção e revisão de software, auxiliando na geração de trechos de código, sugestões de refatoração e correção de erros. |
| **Como?** | Por meio de modelos de deep learning treinados em grandes bases de código, que preveem e sugerem trechos de programação com base no contexto do desenvolvedor. |
| **Por quê?** | Para aumentar a produtividade dos desenvolvedores, reduzir erros, acelerar o desenvolvimento de software e democratizar o acesso à programação para iniciantes. |
